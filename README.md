# Library Management System

A Library Management System is a software application designed to handle the day-to-day administrative tasks of a library efficiently. 
This project provides functionalities to manage books, members, returning operations, and more, serving as an ideal solution for schools, colleges, or community libraries.

---
---
**Member Management**  
  - Register members (Student, Teacher, Guest)  
---
Features

- **Book Management:** Add, update, search, and remove books from the inventory.
- **Member Management:** Register new members, update member details like name, email, phone, and delete member.
- **Return:** Issue books to members and handle returns with due date tracking.
- **Search Functionality:** Search for books and members by various parameters.

## Usage

- **Add Books:** Navigate to the "Books" section and use the "Add Book" form.
- **Register Members:** Use the "Members" section to register new users.
- **Issue/Return Books:** Go to the IssuedTo  section.
- **View Books:** Check the "Books" section for analytics and overdue items.
